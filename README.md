# Pizzaria Java System 🍕

Sistema de gestão de pedidos para pizzarias desenvolvido em Java, com foco em boas práticas de POO, design patterns e persistência de dados.

## Funcionalidades
- **Pedidos Personalizados:** Uso do Padrão Builder para montagem flexível das pizzas.
- **Persistência de Dados:** Registro de pedidos em arquivos de texto.
- **Relatórios:** Cálculo automático de faturamento.
- **Log de Erros:** Monitoramento de falhas do sistema em tempo real.

## Como Executar
1. Certifique-se de ter o Java instalado em sua máquina.
2. Clone o repositório ou baixe o arquivo `.zip`.
3. Compile e execute a classe `Main.java` localizada no pacote `com.pizzasho`.

## Tecnologias Utilizadas
- Java SE
- Git/GitHub
- File I/O (Manipulação de arquivos)
- Exception Handling (Tratamento de erros)
